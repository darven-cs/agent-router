4. initDB function with WAL mode (per 03-RESEARCH.md Pattern 5):
```go
func initDB(dbPath string) (*gorm.DB, error) {
    db, err := gorm.Open(sqlite.Open(dbPath), &gorm.Config{})
    if err != nil {
        return nil, err
    }
    db.Exec("PRAGMA journal_mode=WAL")
    db.Exec("PRAGMA synchronous=NORMAL")
    db.AutoMigrate(&UsageLog{})
    return db, nil
}
```

5. StartUsageWorker function (per D-07, 03-RESEARCH.md Pattern 1):
- Accept db *gorm.DB and usageChan <-chan RequestLog
- Run in goroutine, loop reading from channel
- For each log, create UsageLog and db.Create(&usageLog)
- On error, log to stderr: fmt.Fprintf(os.Stderr, "Usage write error: %v\n", err)
- On success, update in-memory stats: stats.mu.Lock(), increment counters, Unlock()
- Return cleanup func to close channel (not needed for fire-and-forget)

See 03-RESEARCH.md Pattern 1 for exact implementation.
